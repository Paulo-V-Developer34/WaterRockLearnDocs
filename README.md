# Water Rock Learn Docs

Essa é a documentação do [Water Rock Learn (WRL)](https://github.com/Paulo-V-Developer34/WaterRockLearn), que originalmente estava dentro do próprio projeto como uma workspace, mas para facilitar alguns processos foi transferida como um repositório a parte.

## Como executar o Docussaurus

Para executar o Docussaurus primeiro instale as dependências:
```Bash
pnpm i
pnpm approve-builds
```
Agora execute o projeto
```Bash
pnpm start
```
